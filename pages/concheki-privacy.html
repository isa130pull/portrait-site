Concheki プライバシーポリシー

最終更新日: 2026年5月1日

かわぐち（isa130pull）（以下「運営者」といいます）は、スマートフォンアプリ「Concheki」（以下「本アプリ」といいます）におけるユーザーのプライバシーを尊重し、以下のとおり個人情報を適切に取り扱います。

1. 取得する情報

本アプリでは、以下の情報を取得します。

  1. ユーザー識別子: 認証サービス（Firebase Authentication）が発行する匿名 UID
  2. ユーザー入力情報: ユーザーがアプリ内で任意に設定する表示名（最大 16 文字）
  3. 位置情報: チェックインの正当性確認のために、QR コード読み取り時にのみ取得する端末の位置情報(GPS)
  4. 利用ログ: チェックイン履歴、ポイント残高、ガチャ履歴、所持アバター
  5. 端末情報: OS バージョン、端末モデル、アプリのバージョン、言語設定
  6. 利用解析情報: アプリ内の画面遷移、機能の利用頻度等(Firebase Analytics により自動取得)

本アプリは、氏名、電話番号、メールアドレス、住所、生年月日、クレジットカード情報を取得しません。

2. 利用目的

取得した情報は以下の目的に利用します。

  1. 本アプリの機能提供（チェックイン、ポイント付与、ガチャ、アバター管理）
  2. チェックインの正当性確認（位置情報）
  3. 不正利用の検知および防止
  4. 本アプリの品質改善および新機能の検討
  5. 法令に基づく対応

3. 第三者提供

運営者は、以下の場合を除き、取得した情報を第三者に提供しません。

  1. 法令に基づく場合
  2. 人の生命、身体または財産の保護のために必要であり、本人の同意を得ることが困難な場合
  3. 第 4 項に定める業務委託先に、利用目的の達成に必要な範囲で提供する場合

4. 業務委託（外部サービスの利用）

運営者は、本アプリの提供にあたり、以下の外部サービスを利用しています。各サービスのプライバシーポリシーは各社のページをご確認ください。

サービス名 提供事業者 利用目的
Firebase Authentication / Firestore / Cloud Messaging / Analytics / Crashlytics Google LLC 認証、データ保存、通知配信、利用解析、不具合検知
PlayFab Microsoft Corporation ゲームサービス基盤（ポイント・ガチャ抽選・アバター管理）

各社のプライバシーポリシー：

  • Google: https://policies.google.com/privacy
  • Microsoft: https://privacy.microsoft.com/ja-jp/privacystatement

5. Cookie および同等の技術

本アプリは Web ブラウザを介した Cookie の発行は行いませんが、上記第 4 項の外部サービスは、ユーザーの端末上に識別子（広告識別子を含みます）を保存・参照する場合があります。ユーザーは、iOS / Android の各端末設定により、広告識別子のリセットおよびトラッキング許可の制御が可能です。

6. 開示・訂正・削除

ユーザーは、運営者に対して、自身の個人情報の開示、訂正、利用停止、削除を求めることができます。本アプリ内の設定画面からアカウント削除を行うことで、保存された情報は速やかに削除されます。これに加えて手続を希望する場合は、第 9 項のお問い合わせ先までご連絡ください。

7. 安全管理

運営者は、取得した情報の漏えい、滅失、改ざんを防止するため、合理的な安全管理措置を講じます。

8. ポリシーの変更

本ポリシーは、必要に応じて改定されることがあります。改定後のポリシーは、本ページに掲載した時点で効力を生じます。重要な変更がある場合は、本アプリ内で通知します。

10. 外部送信に関する公表（電気通信事業法第 27 条の 12）

本アプリは、ユーザーの利便性向上およびサービスの品質改善のため、ユーザーの端末から外部の事業者へ次の情報を送信しています。

10.1 Firebase Authentication / Firestore / Cloud Messaging（Google LLC）

  • 送信される情報: 匿名 UID、認証トークン、アプリ内データ（表示名、ポイント残高、チェックイン履歴、所持アバター）、デバイストークン
  • 送信目的: ユーザー認証、データの永続化、プッシュ通知配信
  • プライバシーポリシー: https://policies.google.com/privacy
  • 設定変更方法: アプリ内のアカウント削除機能、または端末の通知設定からプッシュ通知をオフにできます

10.2 Firebase Analytics（Google LLC）

  • 送信される情報: アプリ内イベント（画面表示、ボタン押下、チェックイン回数、ガチャ実行回数）、端末情報（OS バージョン、端末モデル、言語設定）、広告識別子（IDFA / AAID。ユーザーが端末設定で許可した場合のみ）
  • 送信目的: アプリの利用状況の解析、品質改善
  • プライバシーポリシー: https://policies.google.com/privacy
  • オプトアウト方法:
    • iOS: 設定 > プライバシーとセキュリティ > トラッキング から本アプリのトラッキングをオフ
    • Android: 設定 > Google > 広告 から「広告のカスタマイズをオプトアウト」を有効化
    • アプリ内のアカウント削除機能でも送信を停止できます

10.3 Firebase Crashlytics（Google LLC）

  • 送信される情報: アプリのクラッシュ発生時のスタックトレース、端末情報、アプリのバージョン
  • 送信目的: アプリの不具合検知および修正
  • プライバシーポリシー: https://policies.google.com/privacy

10.4 PlayFab（Microsoft Corporation）

  • 送信される情報: 匿名 UID、ポイント残高、ガチャ抽選リクエスト、所持アバター ID
  • 送信目的: ポイント・アバターのサーバー管理、ガチャ抽選処理の実行、不正利用の検知
  • プライバシーポリシー: https://privacy.microsoft.com/ja-jp/privacystatement

10.5 公表の更新

本公表内容は、外部送信先または送信内容に変更があった場合、速やかに更新します。
